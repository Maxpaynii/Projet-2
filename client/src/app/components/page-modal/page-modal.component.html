Guide d'utilisation de PolyDessin 11

Pour sélectionner un outil, cliquer sur l'icone de l'outil souhaité. Un menu contenant divers option pour l'outil sélectionné sera affiché.

Annuler-refaire

Permet d'annuler ou de refaire une action faite. Si une nouvelle action est faite, il est impossible de refaire une action annulée.

Sélection

Cliquer ou cliquer-déposer pour sélectionner un groupe de tracés. Une fois un ou des tracés sélectionnés, cliquer sur un des 8 carrés entourant la sélection pour la redimensionner. Utiliser la molette de la souris pour changer l'orientation.

Presse-papier

Permet de copier, couper, coller, dupliquer et supprimer une sélection

Grille

Permet d'afficher une grille. Si l'option de magnétisme est utilisé, les tracés seront attirés vers les lignes de la grille.

Crayon

Maintenir un clic pour effectuer un tracé libre.

Plume

Maintenir un clic pour tracer un tracé libre avec une épaisseur avec une orientation personalisée. Utiliser la molette de la souris pour changer l'orientation.

Pipette

Cliquer pour sélectionner la couleur d'un tracé. Un clic gauche sélectionne la couleur primaire, alors qu'un clic droit sélectionne la couleur secondaire.

Efface

Cliquer sur un tracé pour l'enlever.

Ellipse

Cliquer-déposer pour tracer une ellipse. Maintenir la touche Alt enfoncée permet de tracer un cercle.

Étampe

Cliquer pour ajouter une étampe. Utiliser la molette de la souris pour changer l'orientation.

Aérosol

Maintenir un clic pour parsemer des points à l'endroit du curseur.

Applicateur de couleur

Cliquer sur un tracé pour changer sa couleur.

Texte

Cliquer pour créer une zone de texte.
